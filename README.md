Simulatrix 🏠💡

Simulatrix é um simulador de automação residencial desenvolvido para demonstrar a integração entre aplicações JavaScript e modelos de Inteligência Artificial (LLMs).

Tecnologias
HTML5
CSS3
JavaScript (ES6+)
Node.js
Express
OpenAI API (GPT-4o-mini)
Funcionalidades
Simulação de uma sala com iluminação RGB.
Controle de intensidade e cor das luzes.
Controle de temperatura e estado do ar-condicionado.
Comunicação com um endpoint REST.
Integração com um LLM para interpretar descrições em linguagem natural e gerar automaticamente uma configuração da casa.
Atualização automática da interface a partir de um arquivo de configuração JSON.
Objetivo

Este projeto foi desenvolvido como estudo de integração entre Node.js, Express, APIs REST e Modelos de Linguagem (LLMs), explorando aplicações práticas de Inteligência Artificial em automação residencial.
